# The `factor` command

The `factor` command prints the prime factors of each given number.

### Examples

1. Print prime factors of a prime number

```
factor 53
```

2. Print prime factors of a non-prime number

```
factor 75
```

3. Print prime factors of multiple numbers

```
factor 75 53 29 
```

### Syntax:

```
factor [NUMBER]
```
